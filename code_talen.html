Adobe Dreamweaver

Dit is de hoofdprogramma die ik gebruik op school om de diverse webtalen te programeren. Voorbeelden van deze webtalen zijn: PHP, HTML&CSS, Javascript en jQuery.
Ook weet ik dankzij school hoe ik via Dreamweaver zelf bestanden kan uploaden.

Thuis gebruik ik Atom, omdat het een goede editer is naast Dreamweaver, Atom vind ik ook handiger persoonlijk dan.
